A dynamic GitHub repository visualizer with a premium dark theme. Paste any public repository URL and explore it visually — no installation, no server, just your browser.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 📊 **Dashboard** | Project stats, file types, dependencies, and structure at a glance |
| 🌐 **Live Preview** | Instantly render HTML files in a live preview |
| 📄 **Code Viewer** | Browse code with file preview cards and syntax highlighting |
| 📖 **README** | Beautifully formatted README rendering |
| 🎨 **Premium Theme** | Dark theme with custom orange accent (#E84A00) |
| ⨁ **LaiqVerse Branded** | Integrated with the LaiqVerse identity |

---

## 🚀 How to Use

1. **Open** the tool at: [laiqqureshi.github.io/LaiqVerse-Repo-Explorer/](https://laiqqureshi.github.io/LaiqVerse-Repo-Explorer/)
2. **Paste** any public GitHub URL (e.g., `https://github.com/recloudstream/cloudstream`)
3. **Click** "Load & Visualize"
4. **Explore** the dashboard, files, code, and live preview
